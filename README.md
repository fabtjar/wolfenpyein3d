# Wolfeinpyein 3D

Simple [ray casting]() rendering similar to the [Wolfenstein 3D](https://en.wikipedia.org/wiki/Wolfenstein_3D) engine.

Inspired by javidx9's [video](https://youtu.be/xW8skO7MFYw).

## Usage

```
python3 -m pip install -U pygame --user
python3 game.py
```
